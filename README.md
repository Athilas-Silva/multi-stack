# Adote um Pet

 Sistema de adoção de pets, aplicação desenvolvida na semana Multi-Stack da TreinaWeb. Projeto fullstack, sistema completo para adoções e doações para animais. 
 
 ## Ferramentas utilizadas
 
 - Vs Code
 - Front-End:
   - NextJs
   - Typescript
   - Styled Components
   - Material UI

 - Back-End:
   - Java
   - Spring Boot
